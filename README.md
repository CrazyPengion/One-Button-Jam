As part of the "ONE BUTTON JAM 2026" on Itch.io, this is a game made to only be controlled using a single button. 
Through a combination of holding and tapping [SPACE] you can cast spells to defeat hordes of enemies coming at you, which gives you xp to unlock further spells.

This project was made with the goal of improving my programming skills, which means it was written to try out different features, such as seed based procedural ground texture generation.

With it being a learning project AI was used for: Web support ; Creating an Optimal Pseudo Random Number Generator ; Getting Vector Math Formulas ; General  questions / explanations / debugging

All ideas and art are man-made.
